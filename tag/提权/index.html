                          Mysql UDF提权

                          • UDF（User Defined Function）用户自定义函数：使用过MySQL的人都知道，MySQL有很多内置函数提供给使用者，包括字符串函数、数值函数、日期和时间函数等，给开发人员和使用者带来了很多方便。MySQL的内置函数虽然丰富，但毕竟不能满足所有人的需要，有时候我们需要对表中的数据进行一些处理而内置函数不能满足需要的时候，就需要对MySQL进行一些扩展，幸运的是，MySQL给使用者提供了添加新函数的机制，这种使用者自行添加的MySQL函数就称为UDF(User Define Function)。

                          • 提权条件

                            1. 知道mysql的用户名和密码，且开启了远程连接mysql的功能。
                            2. mysql具有写入文件的权限，即secure_file_priv的值为' '(空)。

                          • 提权效果：跟运行mysql用户有关，如果是管理员用户运行则提到的权限也是管理员。

                          PTT攻击之MS14-068

                          PTT攻击之MS14-068

                          • MS14-068漏洞主要通过伪造域管的TGT，将普通用户权限提权为域管权限，以此来控制域控。只要服务器没有打MS14-068(KB3011780)补丁，在Server 2000以上的域控服务器中，都可以利用。

                          利用条件

                          1. 获取域普通用户的账号密码
                          2. 获取域普通用户的SID
                          3. 服务器未打MS14-068(KB3011780)补丁

                          MS14-068攻击步骤

                          1. 获取用户的SID whoami /all
