                          PHP基础语法

                          PHP语法初步

                          • PHO是一种运行再服务器端的脚本语言，可以嵌入到HTML中。

                          PHP代码标记

                          • 在PHP历史发展中，可以使用多种标记来区分PHP脚本
                          ASP标记：<% PHP代码 %>
短标记：<? PHP代码 ?>
以上两种基本弃用，如果要使用那么需要在配置文件中开启:
php.ini中:
202 short_open_tag = Off #设置为On为开启
206 asp_tags = On #设置为On为开启
    
脚本标记：<script language="php" >php代码</script>>
<html>
    <body>
            <script language="php">
                //脚本标记
                echo 'hello world';
            </script>
    </body>
</html>
                
标准标记(常用)：<?php php代码 ?>
<html>
    <body>
            <?php
                //脚本标记
                echo 'hello world';
            ?>
    </body>
</html>
